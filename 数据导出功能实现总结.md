## 🎯 核心功能

### 数据类型支持

| 数据类型 | 表名 | 支持的过滤条件 |
|---------|------|---------------|
| 日志数据 | log_entries | 日志级别、关键词、时间范围 |
| 告警数据 | alerts | 告警级别、告警类型、状态、时间范围 |
| 安全事件 | unified_security_events | 事件类型、严重程度、时间范围 |
| Windows安全日志 | windows_security_logs | 事件ID、用户名、时间范围 |
| 系统性能指标 | system_metrics | 指标类型、时间范围 |

### 导出格式支持

| 格式 | 扩展名 | 特点 | 适用场景 |
|-----|--------|------|---------|
| CSV | .csv | 文件小、性能好 | 大量数据导出、Excel打开 |
| Excel | .xlsx | 格式美观、支持多工作表 | 报表、数据分析 |
| JSON | .json | 结构化、易于程序处理 | API集成、数据交换 |
| ZIP | .zip | 批量打包 | 批量导出多种数据 |

### 导出模式

#### 单个导出
- 支持详细的过滤条件
- 支持CSV、Excel、JSON三种格式
- 适合精确导出特定数据

#### 批量导出
- 一次性导出多种数据类型
- 支持Excel多工作表或ZIP压缩包
- 适合全量数据备份

### 后端实现

#### 1. 动态查询（Specification）
```java
Specification<LogEntry> spec = (root, query, cb) -> {
    List<Predicate> predicates = new ArrayList<>();
    
    if (startTime != null) {
        predicates.add(cb.greaterThanOrEqualTo(root.get("timestamp"), startTime));
    }
    if (endTime != null) {
        predicates.add(cb.lessThanOrEqualTo(root.get("timestamp"), endTime));
    }
    // ... 更多条件
    
    return cb.and(predicates.toArray(new Predicate[0]));
};
```

#### 2. CSV导出
```java
StringBuilder csv = new StringBuilder();
csv.append("ID,时间戳,日志级别,消息\n");

for (LogEntry log : logs) {
    csv.append(String.format("%d,%s,%s,\"%s\"\n",
            log.getId(),
            formatDateTime(log.getTimestamp()),
            log.getLevel(),
            escapeCsv(log.getMessage())));
}

return new ByteArrayResource(csv.toString().getBytes(StandardCharsets.UTF_8));
```

#### 3. Excel导出
```java
try (Workbook workbook = new XSSFWorkbook(); 
     ByteArrayOutputStream out = new ByteArrayOutputStream()) {
    
    Sheet sheet = workbook.createSheet("日志数据");
    
    // 创建标题行
    Row headerRow = sheet.createRow(0);
    String[] headers = {"ID", "时间戳", "日志级别", "消息"};
    for (int i = 0; i < headers.length; i++) {
        Cell cell = headerRow.createCell(i);
        cell.setCellValue(headers[i]);
        cell.setCellStyle(createHeaderStyle(workbook));
    }
    
    // 填充数据
    int rowNum = 1;
    for (LogEntry log : logs) {
        Row row = sheet.createRow(rowNum++);
        row.createCell(0).setCellValue(log.getId());
        row.createCell(1).setCellValue(formatDateTime(log.getTimestamp()));
        // ...
    }
    
    // 自动调整列宽
    for (int i = 0; i < headers.length; i++) {
        sheet.autoSizeColumn(i);
    }
    
    workbook.write(out);
    return new ByteArrayResource(out.toByteArray());
}
```

#### 5. ZIP批量导出
```java
try (ByteArrayOutputStream baos = new ByteArrayOutputStream();
     ZipOutputStream zos = new ZipOutputStream(baos)) {
    
    for (String dataType : dataTypes) {
        Resource resource = exportData(dataType, ...);
        
        ZipEntry entry = new ZipEntry(dataType + ".csv");
        zos.putNextEntry(entry);
        zos.write(resource.getContentAsByteArray());
        zos.closeEntry();
    }
    
    zos.finish();
    return new ByteArrayResource(baos.toByteArray());
}
```

## 🔄 数据流程

### 单个导出流程
```
用户选择参数 → 前端发送请求 → 后端查询数据 → 格式化数据 → 生成文件 → 返回文件流 → 前端下载
```

### 批量导出流程
```
用户选择数据类型 → 前端发送POST请求 → 后端循环导出 → 打包文件 → 返回压缩包 → 前端下载
```

## 🐛 已知问题

1. **大数据量导出**
   - 问题：超过10万条数据可能导致内存溢出
   - 解决方案：建议分批导出或使用流式处理

2. **Excel列宽**
   - 问题：自动调整列宽可能不够精确
   - 解决方案：可以手动调整或设置固定列宽

3. **中文文件名**
   - 问题：某些浏览器可能不支持中文文件名
   - 解决方案：使用英文文件名或URL编码
