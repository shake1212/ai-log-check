## 🚀 如何使用

### 1. 启动应用
```bash
# 启动前端开发服务器
pnpm dev

# 启动WebSocket服务器（可选，用于真实实时数据）
node websocket-server.js
```

### 2. 访问系统
打开浏览器访问: http://localhost:8000

## 🛠️ 故障排除

### 常见问题

1. **页面空白**
   - 检查开发服务器是否启动
   - 查看浏览器控制台错误信息
   - 确认端口8000未被占用

2. **WebSocket连接失败**
   - 检查WebSocket服务器是否启动
   - 确认端口8080未被占用
   - 系统会自动降级到模拟数据

3. **图表不显示**
   - 检查ECharts依赖是否正确安装
   - 查看浏览器控制台错误信息
   - 确认数据格式正确

4. **导出功能异常**
   - 检查xlsx依赖是否正确安装
   - 确认浏览器支持文件下载
   - 查看控制台错误信息

### 调试模式
- 打开浏览器开发者工具
- 查看Console标签页的错误信息
- 使用Network标签页检查网络请求
- 使用Elements标签页检查DOM结构
