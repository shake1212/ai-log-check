# AI安全日志异常检测系统 - 功能特性

## 🚀 已实现功能

### 1. 📊 ECharts数据可视化图表
- **实时日志流图表**: 使用ECharts实现的动态折线图，支持实时数据更新
- **威胁类型分布图**: 饼图展示不同类型威胁的分布情况
- **系统性能监控图**: 柱状图显示CPU、内存、磁盘、网络使用率
- **图表特性**:
  - 平滑动画效果
  - 交互式提示框
  - 响应式设计
  - 自定义颜色主题

### 2. 🔌 WebSocket实时数据更新
- **实时连接**: 支持WebSocket连接，实现真正的实时数据推送
- **自动重连**: 连接断开时自动重连机制
- **数据同步**: 实时同步日志数据和统计数据
- **连接状态**: 显示连接状态，断开时使用模拟数据
- **模拟服务器**: 提供WebSocket服务器模拟器用于测试

### 3. 📱 响应式布局设计
- **多断点适配**: 支持手机、平板、桌面等多种屏幕尺寸
- **弹性网格**: 使用CSS Grid实现自适应布局
- **移动端优化**: 针对小屏幕设备的特殊优化
- **触摸友好**: 移动端触摸操作优化

### 4. ✨ 动画效果和交互体验
- **页面加载动画**: 淡入淡出效果
- **卡片悬停效果**: 鼠标悬停时的阴影和位移效果
- **按钮动画**: 脉冲动画和点击反馈
- **列表项动画**: 逐项淡入效果
- **图表动画**: ECharts内置的平滑动画

### 5. 📤 数据导出和报表生成
- **多格式导出**: 支持Excel、CSV、JSON格式导出
- **报表生成**: 自动生成Markdown格式的详细报告
- **数据筛选**: 支持按时间、类型、风险等级筛选导出
- **批量操作**: 支持批量导出和报表生成

## 🛠️ 技术栈

### 前端技术
- **React 18**: 现代化的React框架
- **TypeScript**: 类型安全的JavaScript
- **Ant Design**: 企业级UI组件库
- **ECharts**: 强大的数据可视化库
- **Less**: CSS预处理器

### 实时通信
- **WebSocket**: 实时双向通信
- **自定义Hook**: useWebSocket Hook封装

### 数据处理
- **XLSX**: Excel文件处理
- **数据转换**: 多种数据格式转换

## 🚀 快速开始

### 1. 安装依赖
```bash
pnpm install
```

### 2. 启动开发服务器
```bash
pnpm dev
```

### 3. 启动WebSocket服务器（可选）
```bash
node websocket-server.js
```

### 4. 访问应用
打开浏览器访问: http://localhost:8000

## 📊 功能演示

### 仪表盘功能
- **实时数据**: 每3秒自动更新日志数据
- **统计卡片**: 显示系统状态、日志数量、异常事件等
- **图表展示**: 多种图表类型展示不同维度的数据

### 数据导出
- **Excel导出**: 生成.xlsx格式文件
- **CSV导出**: 生成.csv格式文件
- **JSON导出**: 生成.json格式文件
- **报告生成**: 生成Markdown格式的详细报告

## 🔧 配置说明

### WebSocket配置
```typescript
const { isConnected, sendMessage } = useWebSocket({
  url: 'ws://localhost:8080/ws',
  reconnectInterval: 3000,
  maxReconnectAttempts: 5
});
```

