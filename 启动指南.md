# 启动指南

## 环境要求

- Java 21+
- Node.js 18+
- MySQL 8.0
- Python 3.8+（可选，用于日志采集）
- pnpm（前端包管理器）

---

## 1. 数据库初始化

```sql
CREATE DATABASE ai_log_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

执行初始化脚本（按顺序）：

```bash
mysql -u root -p ai_log_system < back-system/database/init.sql
mysql -u root -p ai_log_system < back-system/database/insert_preset_rules.sql
mysql -u root -p ai_log_system < back-system/database/insert_additional_data.sql
```

默认连接配置（`back-system/src/main/resources/application.yml`）：
- 地址：`localhost:3306`
- 数据库：`ai_log_system`
- 用户名：`root`
- 密码：`123456`

---

## 2. 启动后端

```bash
cd back-system
mvn spring-boot:run
```

后端服务地址：`http://localhost:8080/api`

Swagger 文档：`http://localhost:8080/api/swagger-ui/index.html`

---

## 3. 启动前端

```bash
cd ai-log-system
pnpm install
pnpm dev
```

前端地址：`http://localhost:8000`

前端请求代理配置在 `ai-log-system/config/config.ts`，默认代理到 `http://localhost:8080`。

---

## 4. 启动 Python 采集脚本（可选）

```bash
cd back-system/src/scripts
pip install -r ../../requirements.txt
python system_info_collector_v2.py
```

脚本会定期采集系统性能数据并推送到后端 `/api/system-info/ingest`。
