## 🔍 详细分析

### Java后端规则引擎功能

```java
// API端点
POST /api/rule-engine/analyze      // 分析单个事件
POST /api/rule-engine/refresh      // 刷新规则缓存
GET  /api/rule-engine/statistics   // 获取规则统计
```

**规则匹配流程：**
1. 接收安全事件
2. 遍历所有启用的规则
3. 使用对应的匹配器进行匹配
4. 计算威胁分数（0-100分）
5. 确定威胁等级（LOW/MEDIUM/HIGH/CRITICAL）
6. 返回匹配结果

**威胁分数计算模型：**
- 规则匹配分数：40%
- 规则数量加权：30%
- 严重程度加权：30%

### Python规则引擎集成模块

```python
# rule_engine_integration.py 提供的功能

class RuleEngineClient:
    def analyze_event(event: Dict) -> Dict
        # 分析单个事件，返回增强后的事件数据
        
    def analyze_events_batch(events: List[Dict]) -> List[Dict]
        # 批量分析事件
        
    def get_statistics() -> Dict
        # 获取统计信息
```

**增强的事件字段：**
- `threat_score`: 威胁分数（0-100）
- `threat_level`: 威胁等级
- `rule_matched`: 是否匹配规则
- `matched_rule_count`: 匹配的规则数量
- `matched_rules`: 匹配的规则详情列表

## 🛠️ 如何启用规则引擎集成

### 方法1：使用自动补丁脚本（推荐）

```bash
cd back-system/src/scripts

```

这个脚本会自动：
1. 添加规则引擎导入
2. 添加配置选项
3. 初始化规则引擎客户端
4. 在事件收集后调用规则引擎分析
5. 添加统计信息

### 方法2：手动集成

在 `unified_log_collector.py` 中添加：

```python
# 1. 导入规则引擎
try:
    from rule_engine_integration import RuleEngineClient
    RULE_ENGINE_AVAILABLE = True
except ImportError:
    RULE_ENGINE_AVAILABLE = False
    logger.warning("规则引擎集成模块未找到")

# 2. 在配置类中添加
@dataclass
class CollectorConfig:
    # ... 其他配置 ...
    enable_rule_engine: bool = True
    rule_engine_timeout: int = 10

# 3. 在初始化方法中
def __init__(self, config: CollectorConfig = None):
    # ... 其他初始化 ...
    
    # 初始化规则引擎客户端
    self.rule_engine_client = None
    if RULE_ENGINE_AVAILABLE and self.config.enable_rule_engine:
        self.rule_engine_client = RuleEngineClient(
            backend_url=self.config.java_backend_url,
            timeout=self.config.rule_engine_timeout,
            enabled=True
        )

# 4. 在事件收集后分析
def collect_security_logs(self):
    events = []
    # ... 收集事件 ...
    
    # 规则引擎分析
    if self.rule_engine_client:
        for event in events:
            try:
                event = self.rule_engine_client.analyze_event(event)
                self.stats['rule_engine_analyzed'] += 1
                if event.get('rule_matched'):
                    self.stats['rule_engine_matched'] += 1
            except Exception as e:
                logger.warning(f"规则引擎分析失败: {e}")
    
    return events
```
